# Manual do Desenvolvedor

## O que o Desenvolvedor controla
- visão global de salas
- status comercial
- trial
- mensalidade
- plano Free/Básico/PRO
- abertura e bloqueio de sala
- reset de sala
- remoção de sala
- edição rápida dos dados comerciais

## Processo recomendado
1. criar ou localizar a sala
2. revisar responsável e WhatsApp
3. definir plano e status
4. acompanhar vencimentos
5. baixar log quando houver suporte ou incidente
6. manter backups antes de ações destrutivas

## Ações críticas
Antes de remover, resetar ou restaurar:
- baixar backup JSON
- registrar o motivo da ação
- conferir se a sala correta está aberta
