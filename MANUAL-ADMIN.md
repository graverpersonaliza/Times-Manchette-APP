# Manual do Admin

## O que o Admin pode fazer
- entrar na sala com senha admin
- adicionar jogadores
- resetar senha do jogador para `123456`
- marcar presença/ausência
- definir time manualmente
- sortear times
- salvar histórico
- baixar lista
- exportar backup JSON
- restaurar backup JSON
- baixar log de auditoria
- personalizar sala
- agendar partida
- abrir e fechar inscrições
- iniciar nova rodada
- resetar a sala

## Fluxo ideal do Admin
1. entrar no modo Admin
2. abrir a sala correta
3. adicionar jogadores
4. orientar que todos entrem com nome + senha padrão `123456`
5. jogadores criam a própria senha no primeiro acesso
6. conferir presença
7. montar ou sortear times
8. salvar histórico ao final
9. gerar backup periodicamente

## Boas práticas
- não usar a mesma senha admin em todas as salas
- gerar backup antes de resetar sala ou restaurar dados
- usar sempre o mesmo código da sala para manter histórico
- revisar vencimento/plano se a sala for comercial
